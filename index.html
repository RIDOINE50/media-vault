Installation facile

4 étapes simples pour commencer

1

Téléchargez l'APK

Cliquez sur le bouton de téléchargement pour obtenir le fichier APK.

2

Autorisez les sources inconnues

Allez dans Paramètres → Sécurité → Autoriser les sources inconnues.

3

Installez l'application

Ouvrez le fichier APK téléchargé et suivez les instructions d'installation.

4

Profitez !

Lancez BoomMedia et commencez à télécharger vos contenus préférés.

Important :

Si votre téléphone affiche un avertissement de sécurité, cliquez sur "Installer quand même". BoomMedia est une application open source et sûre.
